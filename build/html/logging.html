ロギング¶

次のようにデバッグ時に実行したSQLログを取得することが出来ます。

SQL

TIME

SELECT * FROM user WHERE status=1 LIMIT 5

デバッグモードの設定¶

ちいたんをデバッグモードにします。

<?php
function config_controller( &$c )
{
    $c->SetDebug( TRUE );
}

cheetan_sql_log にログがセットされますので、それを任意の場所に表示するだけです。

ログの表示¶

HTML内でログを表示します。

<?php echo $data['cheetan_sql_log'];?>

ちなみに、出力されるテーブルには cheetan_sql_log というクラスが割り当てられています。 CSSを作成すれば表のデザイン変更が可能です。
