インストールの解説¶

ちいたんは次のステップでインストールできます:

  1. ダウンロードしたファイルを展開します。

  2. サーバーにちいたんのフォルダとファイルをアップロードします。

これで終わりです!

あとはPHPスクリプトからrequireして使用します。
